# SmartCity – Local Business Promotion App

## Overview

SmartCity is an Android mobile application designed to promote local businesses and help users easily find essential services in the Hadapsar region. The application connects customers with nearby service providers such as caterers, plumbers, car repair services, and pesticide control professionals.

The platform improves business visibility and allows service providers to register their businesses and showcase their services to potential customers.

---

## Features

- View local service providers
- Browse services by category
- View business contact information
- Navigate to service location using Google Maps
- Service provider registration
- Location-based service discovery

---

## Services Available

The application currently supports the following categories:

- Catering Services
- Car Repairing
- Pesticide Control
- Plumbing Services

More service categories can be added in future versions.

---

## Technologies Used

- **Programming Language:** Java
- **Frontend:** Android XML Layouts
- **Backend:** Firebase
- **Maps Integration:** Google Maps API
- **Development Environment:** Android Studio

---

## System Architecture

The application supports two types of users:

### 1. Service Seekers
Users who search for nearby services and contact service providers.

### 2. Service Providers
Business owners who register and list their services with details such as:

- Business Name
- Address
- Contact Number
- Service Category
- Location

## Installation

1. Clone the repository

```
git clone https://github.com/yourusername/SmartCity.git
```

2. Open the project in **Android Studio**

3. Connect the project to **Firebase**

4. Add your **Google Maps API Key**

5. Build and run the application on an Android device or emulator
